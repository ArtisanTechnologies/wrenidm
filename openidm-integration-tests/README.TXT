
!!!Warning: integration tests will delete all data from DB and load data used by integration tests.

Requirements: 
- Openidm is up and running
- Openidm is deployed on http://localhost:8080
- No proxy is required to connect to application
- Firefox is installed, default profile will be used
- Path to Firefox has to be defined in environment variable PATH
- Java 1.6 has to be defined in environment variable PATH
- Currently only Windows is supported

For advanced selenium configuration, please visit: 
http://seleniumhq.org/docs/05_selenium_rc.html#specifying-the-firefox-profile

Test execution:
in Windows execute: run.cmd
